Описание курса

Разработка web-сайтов с использованием HTML 5, CSS 3, CMS — это курс по изучению фундаментальных основ разработки сайтов. Направлен на приобретение знаний в области верстки сайтов, установки и настройки CMS систем.

Основная задача курса - научить слушателей создавать качественную HTML разметку, создавать привлекательную, понятную и удобную структуру для посетителей. Полученные смогут применяться как для собственных работ и проектов, так и для работы и трудоустройства.

Минимальные требования к слушателям

  • Уверенное владение персональным компьютером
  • Навыки работы с текстовыми редакторами
  • Знание HTML, CSS
